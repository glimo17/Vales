# Website

Frontend React para mantenimiento CRUD de prestamos.

## Estructura

- src/components
- src/screens
- src/services

## Configurar API

1. Copia `.env.example` a `.env`.
2. Ajusta `VITE_API_URL` si tu API corre en otro puerto.

## Ejecutar

```bash
npm install
npm run dev
```

Abre http://localhost:5173
